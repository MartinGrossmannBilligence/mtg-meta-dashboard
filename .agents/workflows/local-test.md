---
description: How to test the MTG Premodern Lab app locally before pushing to Git.
---

1. Ensure all dependencies are installed:
   ```powershell
   pip install -r requirements.txt
   ```

2. Run the Streamlit app locally:
   ```powershell
   streamlit run app.py
   ```

3. Open your browser at `http://localhost:8501` to verify changes.

4. Check for any errors in the terminal output.

5. Once satisfied, commit and push your changes to Git.
